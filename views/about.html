This is the firts project of the very very recently created Hack'Aveiro collective!

The concept is simple: send text from our webpage or from twitter to the small lcd badges we're wearing!

Basically, it consists of the following parts:

  • 3D Enclosure for the badge
  • Electronic board and LCD to show the messages
  • YUN gateway that connects to get messages from the server and relay them to the badges through R.F.
  • This webserver that provides user interface, queues messages and queries twitter

The endpoints that this server provides for the gateway are:

  • GET /messages/get_one - Gets a message to be transmitted
  • POST /devices/{deviceCode}/ping - Submit a keep-alive ping from a device
  • POST {deviceCode} and {text} to /messages - Submit a new message
